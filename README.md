
# Python Terminal

Uma aplicação para executar scripts Python interativamente no navegador. Ele utiliza [Pyscript](https://pyscript.net/), uma plataforma open source que possibilita rodar scripts Python no navegador.

## Deploy

Para fazer o deploy deste projeto não há nada especial, apenas precisa rodar index.html.

Está executando no [Vercel](https://python-terminal-gamma.vercel.app/) e no [Netlify](https://pythonterm.netlify.app/).

## Tecnologias Utilizadas

HTML, Javascript e um pouquinho de Python
